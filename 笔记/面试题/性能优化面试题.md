# 性能优化面试题

> 前端性能优化高频面试题精选。

---

## 1. 前端性能优化手段

### 加载优化

- 资源压缩（gzip / brotli）
- 图片优化（WebP、懒加载、雪碧图）
- CDN 加速
- 合理使用缓存（强缓存 + 协商缓存）
- 代码分割 + 路由懒加载

### 渲染优化

- 减少回流重绘
- 使用 `transform` 代替 `top/left`
- 虚拟滚动（大列表）
- 防抖节流
- `requestAnimationFrame`

### 代码优化

- Tree Shaking
- 按需引入第三方库
- Web Worker 处理计算密集任务
