# 小程序 & 跨端面试题

> 小程序和 UniApp 跨端开发面试高频考点。

---

## 1. 小程序和 H5 的区别

| 维度 | 小程序 | H5 |
| --- | --- | --- |
| 运行环境 | 微信客户端 | 浏览器 |
| 渲染 | 双线程 | 单线程 |
| DOM | 不支持 | 支持 |
| JS 引擎 | JSCore | V8 等 |
| 入口 | 搜索/扫码/分享 | URL 访问 |
| 审核 | 需要 | 无需 |

---

## 2. setData 性能优化

1. 减少传输数据量（只传变化部分）
2. 减少调用频率（合并多次 setData）
3. 避免在后台页面 setData
4. 避免传输大型数组/对象

---

## 3. 小程序登录流程

```
1. 小程序调用 wx.login() 获取临时 code
2. 将 code 发送到自己的后端服务器
3. 后端用 code + appid + secret 换取 openid + session_key
4. 后端生成自定义登录态（token）返回给小程序
5. 小程序将 token 存储到本地，后续请求携带
```

---

## 4. UniApp 条件编译原理

在编译阶段通过注释标记 `#ifdef` / `#ifndef` / `#endif` 实现代码裁剪，不同平台只保留对应代码，不影响运行时性能。
